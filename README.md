# Practical-work-14

Задача: Реализуйте абстракцию, используя классы. Класс определяет шаблон (абстрактный тип данных) для объекта "Телефон".

1. Создайте класс Phone.

2. В методе __init__ определите атрибуты (данные), которые характеризуют телефон: brand, model, price, color, storage_gb, is_in_stock. Используйте аннотации типов (Type Hints) для каждого атрибута.

3. Создайте методы (функции, работающие с данными объекта), которые предоставят интерфейс для работы с телефоном:
  * get_full_name() -> str: Возвращает строку в формате "Бренд Модель".
  * apply_discount(self, discount_percent: float) -> None: Уменьшает цену телефона на заданный процент.
  * check_availability(self) -> str: Возвращает сообщение "В наличии" или "Нет в наличии" в зависимости от is_in_stock.

4. Создайте метод __str__ для красивого строкового представления объекта (например, при печати через print()).

5. Создайте 3-4 экземпляра класса Phone с разными данными и продемонстрируйте работу всех методов.
